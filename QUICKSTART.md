# 快速开始指南

## 1. 构建项目

在 Xcode 中打开项目：

```bash
open voice-now.xcodeproj
```

选择运行目标为 "My Mac"，然后点击运行按钮（⌘R）。

## 2. 首次运行配置

### 步骤 1: 配置 API Key

1. 应用启动后，点击 "打开设置" 按钮
2. 在 "API Key" 输入框中粘贴您的阿里云百炼 API Key
3. 选择服务区域（推荐：中国大陆（北京））
4. 点击 "保存配置"

### 步骤 2: 授予权限

系统会自动请求以下权限：

#### 麦克风权限

- 系统会弹出对话框请求麦克风访问权限
- 点击 "好" 授予权限

#### 辅助功能权限

- 系统会提示需要辅助功能权限
- 点击 "打开系统偏好设置"
- 在 "隐私与安全性" → "辅助功能" 中
- 找到 Voice Now 并勾选

## 3. 开始使用

### 基本操作

1. **激活识别窗口**

   - 按下键盘上的 **右 Command 键**
   - 会弹出悬浮的麦克风窗口

2. **开始说话**

   - 对着麦克风清晰地说出您想要输入的内容
   - 识别结果会实时显示在窗口中
   - 识别的文本会自动输入到当前焦点的文本框中

3. **关闭识别**
   - 再次按下 **右 Command 键**
   - 或点击窗口中的 "关闭" 按钮

## 4. 使用场景示例

### 场景 1: 在文本编辑器中输入

1. 打开任意文本编辑器（如 TextEdit、VS Code、Notes 等）
2. 将光标定位到要输入文本的位置
3. 按右 Command 键
4. 说话："今天天气很好"
5. 文本会自动输入到编辑器中

### 场景 2: 在浏览器中输入

1. 打开浏览器（Safari、Chrome 等）
2. 点击输入框（如搜索框、评论框）
3. 按右 Command 键
4. 说话："搜索最新的新闻"
5. 文本会自动输入到输入框中

### 场景 3: 在聊天应用中输入

1. 打开聊天应用（如微信、钉钉、Slack 等）
2. 点击消息输入框
3. 按右 Command 键
4. 说出您的消息内容
5. 文本会自动输入，然后按回车发送

## 5. 最佳实践

### 获得更好的识别效果

1. **环境**

   - 在安静的环境中使用
   - 避免背景噪音

2. **说话方式**

   - 保持正常的语速
   - 吐字清晰
   - 不要过快或过慢

3. **麦克风设置**
   - 使用高质量的麦克风
   - 保持适当的距离（15-30cm）
   - 调整系统麦克风音量到合适的级别

### 提高效率的技巧

1. **标点符号**

   - 说 "逗号" 会自动添加 ","
   - 说 "句号" 会自动添加 "。"
   - 说 "问号" 会自动添加 "？"

2. **段落控制**

   - 短暂停顿会被识别为词语间隔
   - 较长停顿会被识别为句子结束

3. **编辑文本**
   - 识别错误时，可以先关闭识别
   - 手动修改错误的部分
   - 再次激活继续输入

## 6. 故障排除

### 问题：无法识别语音

**检查清单：**

- [ ] 麦克风是否正常工作？
  - 在系统设置中测试麦克风
- [ ] 是否授予了麦克风权限？
  - 系统设置 → 隐私与安全性 → 麦克风
- [ ] API Key 是否正确？
  - 在设置中重新检查 API Key
- [ ] 网络连接是否正常？
  - 确保可以访问互联网

### 问题：快捷键不起作用

**检查清单：**

- [ ] 是否授予了辅助功能权限？
  - 系统设置 → 隐私与安全性 → 辅助功能
- [ ] 应用是否在运行？
  - 确保 Voice Now 在后台运行
- [ ] 是否按的是右 Command 键？
  - 左 Command 键不会触发

### 问题：识别结果不准确

**改进方法：**

1. 在更安静的环境中使用
2. 靠近麦克风说话
3. 放慢语速，吐字清晰
4. 使用更好的麦克风设备
5. 调整系统麦克风输入音量

### 问题：文本没有自动输入

**检查清单：**

- [ ] 目标应用是否有文本输入焦点？
  - 确保光标在可输入的位置
- [ ] 是否授予了辅助功能权限？
  - 这是自动输入文本的必要权限
- [ ] 目标应用是否支持文本输入？
  - 某些特殊的输入框可能不支持

## 7. 高级功能

### 连续使用

- 应用会保持 WebSocket 连接
- 可以连续多次使用而无需重新连接
- 60 秒无活动后连接会自动断开

### 自定义配置

在设置页面中，您可以配置：

- API Key
- 服务区域
- 采样率（当前固定为 16000Hz）

## 8. 性能优化

- 首次连接可能需要 1-2 秒
- 后续识别延迟通常在 500ms 以内
- 建议保持应用在后台运行以减少启动时间

## 9. 安全提示

- API Key 存储在本地，不会上传
- 音频数据仅用于实时识别，不会被保存
- 建议定期更换 API Key
