# Voice Now - 实时语音识别工具

这是一个基于阿里云百炼 Fun-ASR 的 macOS 实时语音识别工具。

## 功能特性

- ✨ 全局快捷键激活（右 Command 键）
- 🎤 实时语音识别
- ⌨️ 自动输入识别结果到当前焦点文本框
- 🌐 支持中文、英文、日语等多种语言
- 🎨 美观的悬浮麦克风界面
- ⚙️ 灵活的配置选项

## 系统要求

- macOS 14.0 或更高版本
- 麦克风访问权限
- 辅助功能权限

## 配置步骤

### 1. 获取 API Key

1. 访问 [阿里云百炼控制台](https://help.aliyun.com/zh/model-studio/get-api-key)
2. 创建或获取您的 API Key
3. 注意保存好您的 API Key

### 2. 配置应用

1. 启动 Voice Now 应用
2. 点击 "打开设置" 按钮
3. 输入您的 API Key
4. 选择服务区域：
   - 中国大陆（北京）
   - 国际（新加坡）
5. 点击 "保存配置"

### 3. 授予权限

首次使用时，系统会请求以下权限：

- **麦克风权限**：用于录制您的语音
- **辅助功能权限**：用于监听全局快捷键和输入文本

请在系统设置中授予这些权限：

1. 打开 "系统设置" → "隐私与安全性" → "麦克风"
2. 勾选 Voice Now
3. 打开 "系统设置" → "隐私与安全性" → "辅助功能"
4. 勾选 Voice Now

## 使用方法

1. **激活识别**：按下右 Command 键
2. **开始说话**：对着麦克风清晰地说话
3. **查看结果**：识别结果会实时显示在悬浮窗口中，并自动输入到当前焦点的文本框
4. **关闭识别**：再次按下右 Command 键

## 提示

- 确保在安静的环境中使用，以获得最佳识别效果
- 说话时保持正常的语速和音量
- 识别结果会实时更新，无需等待整句话说完
- 可以在任何支持文本输入的应用中使用

## 技术细节

- **语音识别引擎**：阿里云百炼 Fun-ASR
- **音频格式**：PCM，16kHz 采样率，单声道
- **通信协议**：WebSocket
- **开发语言**：Swift + SwiftUI

## 常见问题

### Q: 无法识别语音

**A:** 请检查：

1. 麦克风是否正常工作
2. 是否已授予麦克风权限
3. API Key 是否正确配置
4. 网络连接是否正常

### Q: 快捷键不起作用

**A:** 请检查：

1. 是否已授予辅助功能权限
2. 在系统设置中确认 Voice Now 已被添加到辅助功能列表

### Q: 识别结果不准确

**A:** 建议：

1. 在安静的环境中使用
2. 靠近麦克风清晰地说话
3. 使用高质量的麦克风设备
